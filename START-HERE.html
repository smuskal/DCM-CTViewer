CT Viewer - Setup Instructions

How It Works

CT Viewer is a simple application that displays your medical scans in 3D. Here's the key concept:

The Software (this folder)

Put this CTViewer-Software folder anywhere on your computer. It contains the viewer application.

Your Scan Images (separate)

Your CT/MRI scan files (.dcm, .jpg, .png) stay wherever they are. The viewer doesn't move or copy them.

Setup for Mac

1

Install Python (if not already installed)

Check if Python is installed by opening Terminal and typing python3 --version

If not installed, click below to download:

Download Python for Mac

After downloading:

  • Double-click the downloaded file (python-3.12.0-macos11.pkg)
  • Click Continue through the installer
  • Click Install and enter your password
  • Click Close when done
2

Run the Start Script

In this CTViewer folder, right-click the file start.sh and select:

Open With → Terminal

First time only: The script will automatically:
  • Create a virtual environment
  • Install all required packages
  • Start the viewer
This may take a few minutes.
If "Open With → Terminal" doesn't appear:
  1. Open Terminal (Command+Space, type Terminal)
  2. Type: cd (with a space)
  3. Drag this CTViewer folder into Terminal
  4. Press Enter
  5. Type: bash start.sh and press Enter
3

Browse and Open Your Scan

Your browser will open to the CT Viewer. You'll see a folder browser:

  • Recent Folders appear at the top - click Open to load directly
  • Or browse to navigate to your scan folder
  • Folders containing images are highlighted in green
  • Click Open This Folder when you find your scan
Next Time: Just right-click start.sh → Open With → Terminal
The virtual environment is saved, so startup will be fast.

Setup for Windows PC

1

Download and Install Python

Download Python for Windows

After downloading:

  • Double-click the downloaded file
  • IMPORTANT: Check the box that says "Add python.exe to PATH" at the bottom of the installer!
  • Click Install Now
  • Click Close when done
2

Launch the Viewer

In this CTViewer-Software folder, double-click the file:

start.bat

This will:

  • Install required components (first time only)
  • Start the CT Viewer
  • Open your web browser automatically
3

Browse and Open Your Scan

Your browser will open to the CT Viewer. You'll see a folder browser:

  • Recent Folders appear at the top - click Open to load directly
  • Or browse through folders to find your scan
  • Folders containing images are highlighted in green
  • Click Open This Folder when you find your scan
Next Time: Just double-click start.bat to launch the viewer. Your recent folders will be remembered!

Using the CT Viewer

Supported File Formats:
  • DICOM (.dcm) - Standard medical imaging format from CT/MRI scanners
  • JPEG (.jpg) - Exported image sequences
  • PNG (.png) - Exported image sequences

Viewing Controls

  • 3D View (left panel):
    • Click and drag to rotate
    • Scroll to zoom in/out
    • Right-click and drag to pan
  • Slice Views (right panel):
    • Use sliders to scroll through slices
    • Axial = top-to-bottom
    • Sagittal = left-to-right
    • Coronal = front-to-back
  • Presets: Use the dropdown to switch between Dental, Bone, Soft Tissue, or Lung views

Troubleshooting

  • "python" not recognized: Restart your computer after installing Python
  • Browser shows "can't connect": Make sure Terminal/Command Prompt is still open
  • To stop the viewer: Close the Terminal/Command Prompt window, or press Ctrl+C
  • Scan not loading: Make sure the folder contains .dcm, .jpg, or .png files (not subfolders)
